## 💡 简介

基于godot4.3版本的回合制战斗系统框架，包含一个独立的技能系统插件。

## ✨ 特性

### 回合制战斗框架

- [ ] 支持自动战斗和手动战斗模式
- [ ] 可切换实时/回合制战斗
- [ ] 灵活的战斗流程控制系统
- [ ] 完整的战斗生命周期事件

### 技能系统插件 (AbilitySystem)

- [ ] 可组合的技能效果系统
- [ ] 灵活的资源管理机制
- [ ] 完整的Buff系统
- [ ] 可扩展的伤害计算系统
- [ ] 多样化的目标选择方式
- [ ] 事件驱动的技能触发机制

## 🚀 快速开始

1. 克隆或下载本项目
2. 将ability_system文件夹复制到你的项目中作为插件使用
3. 参考demo场景和示例代码来实现你自己的战斗系统

## 📁 项目结构

- ability_system/ - 技能系统插件
- combat_system/ - 回合制战斗框架
- demo/ - 示例场景和代码
