# GodotGameplayProjectileSystem

一个灵活且强大的 Godot 4.x 游戏投射物系统

## 特性

- 灵活的投射物系统
- 投射物管理
- 碰撞检测
- 生命周期管理
- 与其他系统的集成

## 快速开始

1. 将 `addons/godot_gameplay_projectile_system` 文件夹复制到你的项目的 `addons` 目录
2. 在项目设置 -> 插件中启用插件
3. 在你的场景中添加 ProjectileSystem 节点
4. 配置投射物场景和属性
